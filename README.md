# SocialApp (Instagram like iOS App)


## App Features ##

- Create account and login
- Create new posts (video and photos)
- Like feature
- Explore top posts (based on numbers of likes)
- Find new people to connect with 
- Update Profile details
- Notifications
- Timestamp Feature
- Follow feauture
- Comment feature
- Hashtags support
- Mentions
